Privacy Policy

YT Transcript - One Click Download · Last updated March 25, 2026

Overview

YT Transcript ("the Extension") is a browser extension that downloads YouTube video transcripts. This policy explains how the Extension handles user data.

Data Collection

The Extension does not collect, transmit, or store any personal data. Specifically:

  • No personal information is collected (name, email, IP address, or otherwise)
  • No analytics, telemetry, or tracking of any kind is used
  • No data is sent to external servers owned or operated by us
  • No cookies are set by the Extension

Data Stored Locally

The Extension uses Chrome's chrome.storage API to save user preferences (such as preferred format or language selection) locally on your device. This data never leaves your browser and is not accessible to us or any third party.

Website Access

The Extension accesses https://www.youtube.com solely to fetch video caption data for the transcript you request. No other websites are accessed. The Extension does not read, modify, or collect your browsing history.

Permissions Used

  • activeTab — to detect when you are on a YouTube video page
  • tabs — to read the current tab URL for auto-detection
  • storage — to save user preferences locally
  • scripting — to execute caption-fetching code within the YouTube page context
  • Host permission for https://www.youtube.com/* — to fetch video captions

Third-Party Services

The Extension does not integrate with or send data to any third-party services, APIs, or analytics platforms.

Changes to This Policy

If this policy is updated, the revised version will be posted at this URL with an updated date.

Contact
